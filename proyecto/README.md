# Prueba de la librería PausablePlayer

## Sobre el programa

Esta es un programa cliente que prueba la librería **PausablePlayer** la cual está basada en la librería **jLayer**.

## Contenido

Este programa contiene las siguientes carpetas y archivos:

	- Makefile: Archivo para compilar el programa.
	- lib/jlayer-1.0.1.jar: Librería jlayer.
	- lib/pausablePlayer.jar: Librería pausablePlayer.
    - mp3Files: Archivos en formato MP3.
	- PruebaPausablePlayer.kt: Programa que prueba la librería pausablePlayer.
	- runPruebaPausablePlayer.sh: Programa para ejecutar el programa PruebaPausablePlayer.kt.
	
## Compilación 

Para compilar el programa ejecute:

`>make`

## Ejecución

Para correr el programa ejecute:

`>runPruebaPausablePlayer.sh archivoMP3`

Donde **archivoMP3** es un archivo de formato MP3.
